== PUT FREERTOS KERNEL FILES HERE ==

Download: https://github.com/FreeRTOS/FreeRTOS-Kernel/releases/latest
Click: "Source code (zip)"

Then copy:
  tasks.c              → freertos/tasks.c
  queue.c              → freertos/queue.c
  list.c               → freertos/list.c
  timers.c             → freertos/timers.c
  event_groups.c       → freertos/event_groups.c

  include/*.h          → freertos/include/   (ALL headers)

  portable/GCC/ARM_CM3/port.c      → freertos/portable/GCC/ARM_CM3/port.c
  portable/GCC/ARM_CM3/portmacro.h → freertos/portable/GCC/ARM_CM3/portmacro.h

  portable/MemMang/heap_4.c        → freertos/portable/MemMang/heap_4.c

Then delete this file and run: make
